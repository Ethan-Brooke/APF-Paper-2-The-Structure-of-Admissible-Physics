## Why this codebase exists

Non-closure theorem: the admissibility algebra admits no single representation valid across all admissibility regimes. Nonabelian carrier ranking (SU(3)×SU(2), with the one-U(1) closure proved a-posteriori for the selected matter content); the conditional 1,680-template classification theorem — relative to the declared representation list, multiplicity caps, and filter inputs, the unique minimum-capacity survivor is the 45-Weyl-fermion Standard Model template; capacity partition C_total = 61 = 45 + 4 + 12, read cosmologically as 42 + 19 (Omega_Lambda = 42/61, Omega_m = 19/61).

This repository is the executable, version-locked certificate layer for those claims.

This is a **version-locked release**: every check function is vendored from the canonical APF codebase at **v24.3.423, commit `5bc6193`** (2026-07-14; 3,912 bank-registered entries, native `verify_all --bank-audit` 3912/3912, gap 0). Each theorem in the manuscript traces to a named `check_*` function in `apf/core.py` or `apf/ec_inventory_reading.py`, callable independently and returning a structured result.

The bundled code requires Python 3.8+ and nothing else — the standard library only.

## The version-locked release and its audit files

The reproducibility contract in Technical Supplement I (v5.1) names five audit files a complete release must carry. They live in [`release_audit/`](release_audit/), generated by `python fermion_scan_standalone.py --emit-audit` at this release's version lock:

| File | Contents |
|---|---|
| `scan_inputs.json` | representation tables, multiplicity caps, filter definitions, equivalence relation, version lock |
| `scan_outputs.csv` | all 1,680 canonical templates (DOF, CPT class) |
| `filter_waterfall.csv` | F1–F7 survivor counts under the canonical F6 predicate, plus the predicate triple as robustness rows |
| `near_misses.csv` | templates failing only the last nontrivial filter — the 42-DOF stripped generation — plus the non-minimal survivors |
| `certificate.sha256` | SHA-256 of each file and of the ordered bundle |

Two further files extend the contract: `kill_index.csv` (per-candidate first-failing filter for every one of the 1,680 templates) and `chirality_audit.csv` (the true-chirality statistic per canonical survivor — the audit-layer report promised at filter F3, reported rather than consumed as a filter).

The canonical F6 predicate is **full-system non-degeneracy** (10 survivors pre-CPT, 5 post-CPT); the spectator-reduction (8/4) and uniform-dimension (4/2) variants are reported as robustness statistics. The winner — the 45-Weyl-fermion Standard Model template — is identical under all three. Waterfall: 1680 → 348 → 324 → 24 → 12 → 10 → 5 → 1.

## About the APF series

The Admissibility Physics Framework is a constraint-first derivation of the Standard Model and cosmological structure from a single primitive — finite enforcement capacity. The corpus runs from the foundational papers through the gauge sector, the quantum formalism, Lorentzian spacetime and the Einstein field equations, the cosmological constant, the electroweak and dark sectors, and the lattice Yang-Mills program. Each paper's main text and Technical Supplement is deposited separately on Zenodo and collected in the **[admissibility_physics](https://zenodo.org/communities/admissibility_physics)** community.
